# VelocityX — Premium Dealership Landing Page

Standalone portfolio project (Assignment A1): luxury car and bike dealership landing page built with HTML, CSS, and Bootstrap 5.

## Pages

- `index.html` — main landing page
- `testdriveform.html` — test drive booking form (Bootstrap modal)

## Run locally

Open `index.html` in your browser (double-click the file, or right-click → Open With → Chrome/Safari). No install, no server, no build step—only HTML, CSS, and Bootstrap.

## Publish on GitHub

1. Create a new repository on GitHub (e.g. `velocityx-landing`).
2. From this folder:

```bash
git init
git add .
git commit -m "Initial commit: VelocityX landing page"
git branch -M main
git remote add origin git@github.com:YOUR_USERNAME/velocityx-landing.git
git push -u origin main
```

3. Optional: enable **GitHub Pages** → Settings → Pages → deploy from `main` / root.

## Stack

- HTML5
- Bootstrap 5 (CDN)
- Custom CSS (`VelocityX.css`)
